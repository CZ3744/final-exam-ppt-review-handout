# Case Study 示例目录

这里用于放置一次真实或脱敏课程 PPT 的完整处理示例。

建议目录结构：

```text
examples/case-study/
├─ README.md
├─ extracted/
│  ├─ sample.compact.md
│  └─ sample.slides.json
├─ analysis/
│  └─ sample.handout.json
├─ outputs/
│  ├─ report.md
│  ├─ report.json
│  ├─ docx/
│  └─ pdf/
└─ notes.md
```

说明：

- 不建议提交未授权的老师原始 PPT。
- 如果 Word/PDF 产物体积较大，建议放到 GitHub Release。
- 公开示例最好使用脱敏课件或自制小样例。
